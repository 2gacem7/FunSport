# FunSport - Back end

## Description
This application is the front of this projet. It's in Vuejs.

## Installation

```bash
$ npm install
```

## Running the app
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
